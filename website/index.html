Noryven regroupe les documents commerciaux, les flux financiers et les rappels d'action dans une interface desktop unique avec rangement par type de document et acces local direct.

  • Edition locale des devis, factures, depenses, recettes et paie

  • Previsualisation des fichiers directement dans l'application

FAQ

Informations utiles.

Les donnees sont-elles dans le cloud ?

Non. Les donnees restent stockees localement sur la machine.

Le telechargement passe-t-il par GitHub ?

Le site redirige vers le dernier binaire publie en release GitHub pour eviter d'alourdir le depot principal.

Puis-je changer les liens plus tard ?

Oui. Il suffit de mettre a jour `website/downloads/manifest.json` et le workflow de build pour publier d'autres noms de fichiers si besoin.
