Pourquoi Velora Finance

Moins de dispersion, plus de controle.

Velora Finance regroupe les documents commerciaux, les flux financiers et les rappels d'action dans une interface claire. Les factures alimente automatiquement le chiffre d'affaires, les depenses influencent la lecture du benefice, et les exports mensuels gardent une trace nette de l'activite.

  • Analyse immediate des finances de l'entreprise
  • Parametres entreprise memorises pour ne pas tout ressaisir
  • Documents HTML stockes en local
  • Organisation par mois pour les factures, devis et depenses
  • Telechargement direct depuis votre site sans passer par la page GitHub

FAQ

Ce qu'il faut savoir avant la mise en ligne.

Les donnees sont-elles dans le cloud ?

Non. Le logiciel enregistre ses donnees localement sur la machine qui l'utilise.

Le telechargement passe-t-il par GitHub ?

Non. Les pages de telechargement de ton site servent directement les binaires publies dans `website/downloads`.

Puis-je changer les liens plus tard ?

Oui. Il suffit de mettre a jour `website/downloads/manifest.json` et le workflow de build pour publier d'autres noms de fichiers si besoin.
