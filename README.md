# Noryven

Noryven est une application desktop locale de gestion financiere pour entreprise.

Fonctions principales :
- tableau de bord financier
- calendrier d'activite
- factures et devis HTML
- recettes, depenses et paie
- todo liste
- stockage SQLite local

Lancement local :
- `py main.py`

Stockage par defaut :
- base SQLite : `%LOCALAPPDATA%\\Noryven\\noryven.db` sous Windows
- racine applicative : `~/.local/share/Noryven` sous Linux
- racine documentaire configurable depuis les parametres
